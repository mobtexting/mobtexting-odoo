MOBtexting SMS Gateway

  • This module use for send the sms from odoo. You need a MOBtexting SMS gateway to setup for trigger sms. You can use multiple sms template. Also you can send the sms manually using wizard or test sms.
  • You can find MOBtexting SMS functionality under the Settings. You will get that menu in debug mode.
  • After debug mode under select the "Become Superuser" you can get MOBtexting SMS Gateway under the Menu Dropdown

GateWay Setup

You can setup your gateway like below.

You should give the Access Token Key, Sender Value and Service Type

country code (like 91 for india)

Test|Manually SMS

This is also use can use send SMS manually.Give the mobile no and message, then click the Send button. This response will update in SMS track with model name mob_gateway_setup.

Template Setup

Setup SMS Template

  • Choose gateway.
  • Choose model.
  • Give the phone no. as a variable like ${object.phone}.
  • Write your message in message area and also you can use replace variables like this {object.name}. . .
  • You can see(* *) right side +Add Contact Action button.if you click this button this SMS template will active in based on your selected Applies to value page. for example i selected "Contact" value so this template will work on contact page.
  • If you don't want to active this template then click -Remove Contact Action button.

SMS Wizard

Go to contact page then click the action button, under this action function you can see your template name.

Also you can modify message content in wizard, but that will not reflect in orginal template.

SMS Track

In the SMS Track page,SMS Template model name like as 'res.partner' will come. Test sms model name like 'mob_gateway_setup' will come

Track History of all message responce. Also that history contain gateway name, phone no, response status and message.
